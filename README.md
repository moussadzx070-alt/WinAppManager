# DZ Manager z
برنامج متقدم وقوي لإدارة وتتبع التطبيقات المثبتة على نظام ويندوز، مع واجهة عصرية حديثة ونظام "الحذف العميق".

## الميزات الجديدة
- **واجهة عصرية (Dark Theme):** تصميم UI/UX احترافي مريح للعين مع تأثيرات بصرية.
- **محرك بحث فوري:** فلترة التطبيقات بالاسم بسرعة فائقة بفضل شريط البحث الذكي.
- **جلب الأيقونات الحقيقية:** قراءة واستخراج أيقونات البرامج الأصلية وعرضها بجانب الاسم.
- **الحذف العميق (Deep Uninstall):** لا يعتمد على ملف الإزالة العادي فحسب، بل يمسح مجلدات `AppData` التابعة للبرنامج ويحذف الاختصارات من سطح المكتب.
- **صلاحيات الإدارة المطلقة:** فرض الإغلاق `Kill` والحذف الإجباري كمسؤول (Administrator).
- **عرض حالة الرام:** مراقبة دقيقة للبرامج النشطة في الخلفية.

## كيفية البناء (Build)
1. ضع أيقونتك في المسار `resources/icon.ico`.
2. قم بتشغيل `build.bat` ليتم تجميع البرنامج باستخدام `PyInstaller`.
3. ستجد البرنامج بصيغة `.exe` جاهزاً للعمل داخل مجلد `dist/DZ_Manager_z`.
