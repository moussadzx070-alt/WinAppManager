# WinAppManager
برنامج لإدارة وتتبع التطبيقات المثبتة على نظام ويندوز، مع إمكانية إزالتها وإيقاف العمليات النشطة الخاصة بها بقوة.

## الميزات
- طلب صلاحيات المسؤول (Administrator) تلقائياً.
- عرض جميع البرامج المثبتة وحجمها.
- تتبع استهلاك الرام (RAM) لكل برنامج.
- إظهار حالة البرنامج (Active بالأخضر، Inactive بالأحمر).
- زر `Uninstall` لحذف البرنامج من النظام.
- زر `إيقاف (Kill)` لفرض إيقاف البرامج التي تعمل في الخلفية.

## كيفية البناء (Build)
1. ضع أيقونتك في المسار `resources/icon.ico`.
2. قم بتشغيل `build.bat` ليتم تجميع البرنامج باستخدام `PyInstaller`.
3. ستجد البرنامج بصيغة `.exe` داخل مجلد `dist/WinAppManager`.
